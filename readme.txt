Compress and download plugins/themes from your wordpress website

== Description ==

If you want to download a plugin or theme from your wordpress website, you should open file manager from hosting.
But with hooker you can download your themes and plugins by just select them.

== Installation ==

1. Extract the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After activating the plugin, in top of the wordpress dashboard menu you can see 'Hooker'. From this menu you can choose what plugin/theme you want to compress and download it.
